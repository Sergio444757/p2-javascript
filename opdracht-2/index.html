Opdracht 2 - Onderzoeksvragen

1. Wat is het verschil met de script tag boven in te laden in de <head> tag t.o.v. onder inladen aan het einde van je HTML tag?

Wanneer je een <script>-tag in de <head>-sectie plaatst, wordt het script geladen en uitgevoerd vóórdat de pagina volledig is weergegeven. Dit kan ertoe leiden dat de pagina trager lijkt, omdat de browser moet wachten tot het script klaar is. Als je de <script>-tag onderaan de pagina zet, vlak voor </body>, wordt eerst de HTML geladen en daarna pas je JavaScript. Hierdoor wordt de pagina sneller zichtbaar voor de gebruiker.

2. Wat doet het attribuut "defer" in de <script> tag?

Het defer-attribuut zorgt ervoor dat het script wordt gedownload terwijl de HTML verder geladen wordt, maar pas wordt uitgevoerd nadat de volledige HTML-pagina is ingeladen. Dit voorkomt blokkeringen in het laden van de pagina en zorgt ervoor dat scripts in de juiste volgorde worden uitgevoerd.

3. Wat is het verschil tussen HTML en JavaScript?

HTML (HyperText Markup Language) wordt gebruikt om de structuur en inhoud van een webpagina vast te leggen, zoals tekst, afbeeldingen en knoppen. JavaScript is een programmeertaal die wordt gebruikt om webpagina's interactief en dynamisch te maken, bijvoorbeeld om animaties te maken, knoppen te laten reageren of gegevens op te halen.

Bonus: Wat is het verschil tussen innerHTML en innerText?

innerHTML geeft HTML inclusief tags terug of voegt het toe, terwijl innerText alleen de zichtbare tekst toont en geen HTML-tags verwerkt. InnerHTML kan dus HTML-elementen toevoegen, innerText werkt alleen met tekst.
